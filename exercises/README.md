# C语言练习题集

这个目录包含了C语言各个常见章节的练习题。每个文件对应一个主题，包含若干个练习函数。

## 目录结构

- `01_intro.c`: 简介与 Hello World
- `02_variables_datatypes.c`: 变量与数据类型
- `03_operators.c`: 运算符
- `04_control_flow.c`: 控制流 (if, switch)
- `05_loops.c`: 循环 (for, while)
- `06_functions.c`: 函数
- `07_arrays.c`: 数组
- `08_pointers.c`: 指针
- `09_strings.c`: 字符串
- `10_structs.c`: 结构体
- `11_file_io.c`: 文件读写

## 如何使用

1. 打开对应的 `.c` 文件。
2. 阅读练习题描述。
3. 在函数体中实现代码。
4. 编译并运行程序以验证结果。

例如，使用 gcc 编译第一个练习：

```bash
gcc 01_intro.c -o intro
./intro
```
